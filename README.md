## ğŸ’» Usage Guide

### ğŸ¯ Step-by-Step Tutorial

#### **Step 1: Launch Application**
```bash
java EnhancedResumeBuilder
```

#### **Step 2: Fill Personal Information**
1. Navigate to **"Personal Info"** tab
2. Enter your **name** (required), **email** (required), and **phone** (required)
3. Add optional fields: **address**, **LinkedIn**, **website**
4. Click photo placeholder to upload profile picture (optional)

#### **Step 3: Add Resume Content**
1. Switch to **"Resume Content"** tab
2. Fill in the following sections:
   - **Professional Summary**: Brief overview of your career
   - **Work Experience**: Job titles, companies, dates, responsibilities
   - **Education**: Degrees, institutions, graduation dates
   - **Skills**: Technical and soft skills
   - **Projects**: Notable projects with descriptions
   - **Certifications**: Professional certifications and courses

#### **Step 4: Customize Design**
1. Go to **"Design & Layout"** tab
2. Select a **template** from 13 options
3. Choose **font size** (10pt - 14pt)
4. Pick a **color scheme** from 12 options
5. Adjust **margins** using slider (10-50)
6. Set **line spacing** (1.0 - 3.0)

#### **Step 5: Preview & Generate**
1. Click **"Live Preview"** to see your resume
2. Make adjustments as needed
3. Click **"Generate PDF Resume"** to export
4. Your resume will be saved with timestamp naming

### ğŸ¯ Pro Tips

- **Save Templates**: Click "Save Template" to reuse later
- **Load Templates**: Click "Load Template" to restore saved data
- **Clear All**: Use "Clear All" button to start fresh
- **Validation**: Required fields are marked with *
- **Export Format**: Resumes are saved as formatted text files

- **Input Validation**: Email format, required fields
- **Template Building**: StringBuilder pattern for content
- **File I/O**: Save/load template functionality
- **Date Formatting**: SimpleDateFormat for timestamps

- **Custom Colors**: Professional color palette

### Quick Reference

#### **Keyboard Shortcuts**
- `Ctrl+S` - Save template
- `Ctrl+L` - Load template
- `Ctrl+P` - Preview resume
- `Ctrl+G` - Generate resume
- `Ctrl+N` - Clear all

#### **File Formats**
- Templates: `.txt` format
- Exports: `.txt` format (PDF coming soon)
- Images: `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`

#### **System Requirements**
- **OS**: Windows, macOS, Linux
- **Java**: JDK 8 or higher
- **RAM**: 512 MB minimum
- **Storage**: 50 MB free space
